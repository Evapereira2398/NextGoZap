# FrontEndNextGoZap

## MOdificar o backend

Para alterar o host e a porta, basta acessar o arquivo [config.js](/public/config.js).

E alterar o valor de **window.IP_SERVER** & **window.IP_SOCKET_IO**.

## Iniciar

```jsx
yarn start
//ou
npm start
```
